# Bahamut Anime Comment Mountains

彈幕密度山脈 for 巴哈姆特動畫瘋

## 功能

-   縱觀整集彈幕分佈
-   提前知道什麼時候會被彈幕洗版

## 安裝

1. 請先安裝 Tampermonkey 或其他網頁腳本管理工具
2. 下載 [dist/index.user.js](https://github.com/JacobLinCool/Bahamut-Anime-Comment-Mountains/raw/main/dist/index.user.js)
